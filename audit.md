📖 Getting Started

Welcome to PromptForge — the multi-surface AI prompt engine.
With PromptForge, you can upgrade prompts with AI, manage usage via credits, and deploy across VS Code, Chrome, and API.

🚀 Quick Start

Get productive in minutes:

Sign up for a free account

Authenticate via Google or Email/Password

Open the Dashboard → view credits & account info

Use the “Upgrade Prompt” button in Chrome/VS Code or via API

🔑 Authentication

PromptForge uses Firebase Authentication with:

Google OAuth

Email/Password

API access via Bearer token

Tokens are validated on every request by our backend.
💡 Tip: Refresh tokens automatically handled in Chrome & VS Code extensions.

✍️ Your First Prompt

Navigate to Dashboard → Prompt Upgrade

Enter your text prompt

Click Upgrade with Oracle

View the upgraded version instantly

Check your credits balance (auto-deducted per upgrade)

🔮 Features
Oracle (✅ working)

Upgrade and refine prompts instantly with AI.
Available via Dashboard, Chrome Extension, VS Code Extension, and API.

Credits & Billing (✅ working)

Every prompt upgrade costs credits.
Top-up via Paddle/Razorpay. Credits auto-sync across devices.

Analytics (⚠️ beta)

Track how many prompts you’ve forged and monitor credit usage.
More dashboards coming soon.

Deploy Anywhere (✅ working)

Chrome Extension (context menu + popup)

VS Code Extension (CodeLens + hover suggestions)

REST API (integrations ready)

📡 API Reference

Base URL (dev): http://localhost:8080/api/v1
Base URL (prod): https://api.promptforge.ai/v1

Auth: POST /users/auth/complete

Profile: GET /users/me

Upgrade Prompt: POST /prompt/upgrade

Credits: GET /users/credits

Analytics: POST /analytics/events

🔒 Auth required for all except /health.

🔌 Integrations

Chrome Extension → upgrade prompts directly from your browser

VS Code Extension → inline upgrade via CodeLens & hover

API → integrate into your workflows, apps, or backends

🔔 Webhooks

Paddle events (subscription, payments)

Webhook docs: /api/v1/payments/webhooks/*
